What is HTML?

HTML is the standard markup language used to create web pages and its elements form the building blocks of all websites.

What are APIs?

An Application Programming Interface (API) is a software inter face that exposes access to some internal functionality of a piece of software for use by programmers, to get access to specific information, to trigger speci al behavior, or to perform some other action. The APIs that the WebApps WG dev elops are typically client-side script APIs, for use in browsers and similar us er agents (as opposed to server-side APIs, for example).
